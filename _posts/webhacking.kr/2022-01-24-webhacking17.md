## 문제 풀이
   화면의 check 버튼을 누르면 sub() 함수가 작동하고 sub함수는 다음과 같이 정의 되어 있습니다.

  ```javascript
  function sub(){ 
    if(login.pw.value==unlock){ 
      location.href="?"+unlock/10; 
      } 
    else{ 
      alert("Wrong"); 
      } }

  ```

  화면상에 복잡하게 되어있는 unlock 값을 입력하면 됩니다.
